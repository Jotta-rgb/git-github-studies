#  git-github-studies

##  Sobre
Repositório criado para centralizar minhas anotações, comandos essenciais e guias práticos sobre o uso do Git (controle de versão) e do GitHub (plataforma de hospedagem e colaboração).

##  Objetivo
Dominar o fluxo de trabalho de versionamento de código, entender a manipulação de históricos, gerenciamento de branches e aplicar as melhores práticas de commits do mercado.

##  Conteúdo Programado
- `comandos-git`: Principais comandos para o dia a dia no terminal.
- `branches`: Fluxos de ramificação (criação, navegação e organização).
- `merge-e-rebase`: Resolução de conflitos e integração de códigos.
- `boas-pratcas`: Padrões de Conventional Commits e organização de repositórios.
- `anotacoes`: Dicas rápidas e resumos teóricos.

##  Ferramentas Utilizadas
- Git (Terminal Bash)
- GitHub
